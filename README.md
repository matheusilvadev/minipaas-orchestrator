### 🚧 Implementação em Andamento
O ZeroPaaS é uma plataforma de serviço (PaaS) minimalista desenvolvida para automatizar o ciclo de vida de aplicações (Build & Runtime) em servidores on-premises. O projeto foca em transformar um servidor caseiro em um ambiente de deploy automatizado.

## 📍 Status Atual
Atualmente trabalhando na integração entre a camada de Aplicação (orquestração assíncrona) e a Infraestrutura (Docker, Git e Networking) para validar o fluxo ponta a ponta:
Código -> Build -> Imagem -> Container -> Runtime.
